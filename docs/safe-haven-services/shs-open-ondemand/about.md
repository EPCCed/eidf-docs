# About the SHS Open OnDemand service

The SHS Open OnDemand service is a web service that runs within a safe haven. The service provides a suite of apps that allows you to run compute and data-related tasks and packages on compute resources available to your safe haven.

![Open OnDemand home page](../../images/open-ondemand/home-page.png){: class="border-img center"} *Open OnDemand home page*

To quickly get started with the SHS Open OnDemand Service (hereon referred to as Open OnDemand for brevity):

* [Access Open OnDemand](access.md).
* Follow [Getting started](getting-started.md) to walk through the key points of the service.

!!! Note

    The SHS Open OnDemand service is implemented using [Open OnDemand](https://openondemand.org/).
